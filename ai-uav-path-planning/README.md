# 无人机动态避障路径规划模拟系统（修正版）

本版本只保留课程设计实验代码，不包含 Word 报告文件。

## 主要修改

1. **算法对比实验固定为三算法对比**：Dijkstra、A*、改进A* 同时运行，不再区分“只比较 A* 和改进A*”与“三算法扩展对比”。
2. **导出功能重新整理**：
   - 不再导出路径地图图片，路径图建议直接使用 GUI 运行截图；
   - 当前导出：导出当前这一次单算法/三算法对比/动态避障的 CSV 和 PNG 分析图；
   - 批量导出：自动运行内置三组静态实验，导出 9 条记录的 CSV 和 PNG 条形图。
3. **内置批量实验明确化**：普通障碍、密集障碍、高密障碍是程序内置的 3 个固定随机场景，不需要在界面中额外选择障碍物密度。
4. **时间统计口径统一**：运行时间为多次重复搜索的平均算法耗时，不包含搜索动画、无人机飞行动画、表格弹窗和图表导出时间。
5. **PNG 图表导出**：如果安装了 matplotlib，会导出路径长度、转折次数、搜索格数、扩展状态数、综合代价、平均运行时间 6 张 PNG 条形图；如果无法生成 PNG/JPG，程序会自动跳过图片，只保留 CSV。

## 运行方式

```bash
python main.py
```

## 批量实验命令行运行

```bash
python experiments.py
```

输出目录为 `results/`。

## 指标说明

- 路径长度：最终路径移动步数，等于 `len(path)-1`。
- 转折次数：路径方向发生变化的次数。
- 综合代价：路径长度 + 转折次数 × 转弯惩罚系数。
- 搜索格数：可视化层面的唯一搜索网格数量。
- 扩展状态数：算法内部实际弹出的状态数量。改进 A* 使用“位置 + 进入方向”作为状态，因此可能与搜索格数不同。
- 平均运行时间/ms：多次重复搜索的平均算法耗时，不包含界面动画时间。
