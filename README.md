# Course Projects Collection

This repository contains sanitized course-project code and experiment assets. It keeps runnable source code, test cases, requirements, flowcharts, and experiment figures. It intentionally excludes Word/PDF reports, paper-writing materials, generated executables, dependency folders, temporary files, and personal information.

## Projects

### ai-uav-path-planning
Python implementation and experiment materials for a UAV dynamic obstacle-avoidance path-planning simulation based on an improved A* algorithm.

Included:
- Algorithm and experiment source code
- Python dependency list
- Experiment CSV outputs
- Experiment result charts
- System/algorithm diagrams

### compiler-pl0-extensions
C implementation materials for a PL/0 compiler course-design project with selected language extensions.

Included:
- Final C source and header files
- Test cases
- Running notes
- Flowchart images

## Privacy And Publishing Notes

Excluded by default:
- Word, PDF, and Office report files
- Paper writing drafts and report-generation scripts
- Generated executables
- Dependency folders and caches
- Temporary files and backups
- Files that may contain names, student IDs, contact information, or document metadata
